# SPECTER INC. Telegram WebApp

React + TypeScript SPA styled у дусі darktrader.co. Готова до деплою як Telegram Mini App.

## Вимоги

- Node.js 18+

## Встановлення

```bash
npm install
```

## Режим розробки

```bash
npm run dev
```

## Білд

```bash
npm run build
```

## Telegram WebApp

Задеплойте вміст `dist/` на HTTPS, потім у BotFather використайте `/setdomain` та додайте WebApp кнопку/меню для вашого бота.
